- 使用CocoaPods(AFN为例)

1. cd .xcodeproj的上一级文件夹

2. pod init

3. 把创建的Podfile文件 拖到 Xcode图标里面

4. ```
   # Uncomment the next line to define a global platform for your project
   # platform :ios, '9.0'
   
   target 'OC单例' do
    pod 'AFNetworking'
   end
   ```

5. 安装框架 pod install

6. ```bash
   # 今后升级、添加、删除框架，或者框架不好用
   $ pod update
   ```

```
